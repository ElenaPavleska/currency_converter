2. Setup
---------
To setup the application simply follow the steps in this guide.

###Prerequisites###
Make sure your have [nodeJS](https://nodejs.org/en/) installed.

###Installation###
**Step by step**

1. `npm install -g grunt`
2. `npm install`
3. `bower install`
4. `grunt` to run the app with browsersync
Have fun!
